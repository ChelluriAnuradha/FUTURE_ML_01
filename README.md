# FUTURE_ML_01
# Sales & Demand Forecasting for Businesses

## Project Overview

This project predicts future sales using historical sales data and Machine Learning techniques.

## Objective

To help businesses:

* Plan inventory
* Manage staffing
* Improve budgeting
* Forecast future demand

## Dataset

Superstore Sales Dataset

## Tools Used

* Python
* Google Colab
* Pandas
* Matplotlib
* Prophet

## Methodology

1. Data Cleaning
2. Exploratory Data Analysis
3. Time-Series Preparation
4. Sales Forecasting using Prophet
5. Model Evaluation

## Results

* Generated a 30-day sales forecast
* Identified trends and seasonality
* Achieved MAE of 1962.20

## Business Impact

The forecast can help businesses optimize inventory levels, reduce stock shortages, and improve operational planning.
